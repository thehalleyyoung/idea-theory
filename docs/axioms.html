Axioms of Idea Theory

Idea Theory is built on a small, purely algebraic signature. The interpretive content of Volumes II–VI is added on top of these axioms; the axioms themselves make no extra-mathematical commitments.

Signature

namespace IdeaTheory

variable (I : Type*)               -- the carrier of ideas
variable [Monoid I]                -- composition  ∘ : I × I → I
variable (R : I → I → ℝ)           -- resonance pairing
variable (w : I → ℝ)               -- weight / norm
variable (κ : I → I → I → ℝ)       -- emergence 2-cocycle
variable (γ : I → I → ℝ)           -- meaning curvature
variable (g : I → ℕ)               -- grading

Core axioms

NameStatementUsed in Theorem
A1 Composition(I, ∘, 1) is a monoid.T1
A2 BilinearityR is bilinear over the free real module on I.T2
A3 Aufhebung splitEvery x ∈ I admits a decomposition x = π(x) ∘ ν(x) ∘ σ(x) (preserved, negated, sublated parts).T3
A4 Cocycle conditionκ(a,b,c) − κ(a∘b,c,d) + κ(a,b∘c,d) − κ(a,b,c∘d) = 0.T4
A5 Curvatureγ(a,b) = R(a,b) − R(b,a) is antisymmetric.T5
A6 ConjugationFor each g ∈ I, x ↦ g ∘ x ∘ g⁻¹ (when defined) is a weight-preserving automorphism on its image.T6
A7 ChainsFinite sequences (xᵢ) with linking morphisms form transmission chains; weight propagates additively up to a curvature defect.T7
A8 EquivalenceStructural equivalence ~ is the smallest congruence containing conjugation and trivial-cocycle moves.T8
A9 Gradingg(x ∘ y) = g(x) + g(y); R respects the grading.T9

Literature bridge

The axioms are deliberately minimal so that informal claims from the literature translate cleanly. See each application volume for the bridge tables; key sources include Hegel (Wissenschaft der Logik) for Aufhebung, Lotman for the semiosphere, Sperber for the epidemiology of representations, Fodor & Pylyshyn for compositionality, and Bedau / Humphreys for emergence.

All axioms are formalised in IdeaTheory/Foundations.lean and referenced by the nine theorem files.
