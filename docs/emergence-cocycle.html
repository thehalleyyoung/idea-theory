emergence cocycle

Home · Theorems · Emergence Cocycle
The emergence 2-cocycle ε : I × I → A measures the genuinely novel content produced when two ideas compose: the part of a ∙ b that is not already determined by a and b separately. Theorem 4 shows that ε satisfies the cocycle identity ε(a, b∙c) + ε(b, c) = ε(a∙b, c) + ε(a, b), which underwrites Volume V's formal account of the “problem” of emergence.

Definition

structure Emergence (M : IdeaMonoid) (A : AddCommGroup) where
  ε : M.carrier → M.carrier → A
  cocycle : ∀ a b c, ε a (M.compose b c) + ε b c = ε (M.compose a b) c + ε a b

Headline theorem

Theorem 4 (Cocycle). Every Aufhebung-compatible emergence map is a normalized 2-cocycle on the idea monoid with values in A. ✓ verified
