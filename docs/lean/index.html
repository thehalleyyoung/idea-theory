Lean Formalization

Machine-verified proofs of Idea Theory in Lean 4 with Mathlib — zero sorry, zero admit.

Why Lean 4?

Lean 4 is an interactive theorem prover and dependently typed programming language developed at Microsoft Research and AWS. Its accompanying mathematics library, Mathlib, contains tens of thousands of machine-verified results spanning algebra, topology, analysis, combinatorics, and game theory. By building Idea Theory on Lean 4 + Mathlib, every claim in the formalism inherits the full weight of that verification infrastructure.

The formalization achieves three things simultaneously. First, it forces complete precision: every definition must be stated without ambiguity, every hypothesis made explicit. Second, it provides a check against the informal theory: mismatches between the mathematical prose and the Lean code reveal hidden assumptions. Third, it creates a reusable artifact — downstream work in economics, linguistics, or cognitive science can import these modules and build on them with confidence.

All six modules compile cleanly with lake build. There are no sorry or admit placeholders anywhere in the codebase — every theorem is fully proved by Lean's kernel.

Architecture

The dependency graph is simple. Foundations stands alone and defines the IdeaTheoryStructure type class on which everything else is built. The five extension modules each import Foundations (and GameTheory also imports GameTheory in the case of MeaningGames). No circular dependencies exist. 

-- Dependency graph (simplified)
IdeaTheory.Foundations
  ├── IdeaTheory.GameTheory
  │     └── IdeaTheory.MeaningGames
  ├── IdeaTheory.Diffusion
  ├── IdeaTheory.RecombinantGrowth
  └── IdeaTheory.EmergenceLevels

Browsing the Source

The full Lean source lives in the lean/IdeaTheory/ directory of the GitHub repository. Each file is self-contained with doc-string comments explaining the mathematical motivation. To build locally: 

git clone https://github.com/thehalleyyoung/idea-theory.git
cd idea-theory/lean
lake update   # fetches Mathlib
lake build    # compiles all modules (~5–10 min first time)

Lean 4 with Mathlib can also be browsed interactively in VS Code using the lean4 extension, which provides hover-over goals, auto-completion, and inline error messages. The Mathlib community site hosts additional documentation.
