Diffusion

Lean 4 Extension — Cyclic Population Transmission

This extension formalises a discrete-time model of idea diffusion through a finite cyclic population. Agents are arranged in a ring; at each step every agent adopts the \(\circ\)-composition of its left neighbour's idea with its own. Thirty-four theorems certify the model's core properties: population conservation, identity fixed points, semigroup iteration, aggregate stability, monotone adoption, and the Granovetter strict-increase effect.

Source: IdeaTheory/Extensions/Diffusion.lean

Model Overview

A population is a List α of ideas drawn from the idea monoid \((\mathcal{I},\circ,e)\). The function step applies one round of cyclic transmission: each agent at position \(i\) adopts \(\mathrm{idea}_{i-1} \circ \mathrm{idea}_i\), with the left-most agent receiving from the right-most (cyclic boundary). The helper lastEl x xs retrieves the terminal element of a list (defaulting to x on the empty list), implementing the cyclic wraparound. 

-- Population model: cyclic transmission step
abbrev Population (α : Type*) := List α

def lastEl (x : α) : List α → α
  | []      => x
  | y :: ys => lastEl y ys

def stepWith (prev : α) : List α → List α
  | []      => []
  | x :: xs => op prev x :: stepWith x xs

def step : List α → List α
  | []      => []
  | y :: ys => stepWith (lastEl y ys) (y :: ys)

Given these primitives, three derived operations complete the model:

  • \(\mathtt{iterate}\;n\;(\mathtt{step})\;\mathit{xs}\) — \(n\) rounds of transmission, written \(\mathtt{step}^{[n]}\;\mathit{xs}\);
  • \(\mathtt{aggregate}\;\mathit{xs} = x_0 \circ x_1 \circ \cdots \circ x_{k-1}\) — the \(\circ\)-fold of the population;
  • \(\mathtt{propagated}(a, L, n) = \mathtt{step}^{[n]}(\mathtt{replicate}\;L\;a)\) — the state of an \(L\)-agent all-\(a\) population after \(n\) steps.
-- T1/T2: Length preservation and identity fixed point
theorem step_length (xs : List α) : (step xs).length = xs.length
theorem step_replicate_ident (n : Nat) :
    step (List.replicate n (ident : α)) = List.replicate n ident

-- T3/T4: Semigroup law and convergence
theorem iterate_step_add (n m : Nat) (xs : List α) :
    step^[n + m] xs = step^[n] (step^[m] xs)
theorem propagated_ident (L n : Nat) :
    propagated (ident : α) L n = List.replicate L ident

-- T8/T9: Rogers monotonicity and Granovetter weak tie
theorem rogers_monotone [DecidableEq α] {f : List α → List α}
    (hf : NonErasing f) (xs : List α) (n : Nat) :
    adopters xs ≤ adopters (f^[n] xs)
theorem granovetter_weak_tie [DecidableEq α] (a : α) (ha : a ≠ ident) (j : Nat) :
    adopters (a :: List.replicate (j+1) (ident : α))
      < adopters (step (a :: List.replicate (j+1) ident))

§1 — Structural Lemmas on lastEl and stepWith

The first six results are definitional unfoldings; they establish the computational behaviour of lastEl and stepWith on nil and cons lists. All proofs close with rfl.

Theorem (lastEl_nil). For any default element \(x\), \(\mathtt{lastEl}\;x\;[] = x\).
Proof sketch: by definition (rfl).
Theorem (lastEl_cons). \(\mathtt{lastEl}\;x\;(y{::}ys) = \mathtt{lastEl}\;y\;ys\).
Proof sketch: by definition (rfl).
Theorem (stepWith_nil). \(\mathtt{stepWith}\;p\;[] = []\).
Proof sketch: by definition (rfl).
Theorem (stepWith_cons). \(\mathtt{stepWith}\;p\;(x{::}xs) = (p \circ x) :: \mathtt{stepWith}\;x\;xs\).
Proof sketch: by definition (rfl).
Theorem (step_nil). \(\mathtt{step}\;[] = []\).
Proof sketch: by definition (rfl).
Theorem (step_cons). \(\mathtt{step}(y{::}ys) = \mathtt{stepWith}(\mathtt{lastEl}\;y\;ys)(y{::}ys)\).
Proof sketch: by definition (rfl).

§2 — Length Preservation

Diffusion is a closed system: no agent is created or destroyed. The following three theorems certify this invariant at the level of a single step, a single iterate, and all iterates.

Theorem (stepWith_length). \(|\mathtt{stepWith}\;p\;\mathit{xs}| = |\mathit{xs}|\).
Proof sketch: Induction on xs; base rfl; step simp using IH.
Theorem (step_length). For all populations \(\mathit{xs}\), \[|\mathtt{step}(\mathit{xs})| = |\mathit{xs}|.\]
Proof sketch: Cases on xs; nil trivial; cons follows from stepWith_length.
Theorem (iterate_step_length). For all \(n\), \[|\mathtt{step}^{[n]}(\mathit{xs})| = |\mathit{xs}|.\]
Proof sketch: Induction on n; step case combines step_length and IH.

§3 — All-Identity Fixed Point

When every agent holds the identity idea \(e\), the population is algebraically inert: transmission of \(e\) by composition with \(e\) reproduces \(e\). The theorems below certify that this state is a fixed point at every level of abstraction.

Theorem (stepWith_replicate_ident). \(\mathtt{stepWith}\;e\;(\mathtt{replicate}\;n\;e) = \mathtt{replicate}\;n\;e\).
Proof sketch: Induction; step uses id_left and IH.
Theorem (lastEl_replicate_self). \(\mathtt{lastEl}\;a\;(\mathtt{replicate}\;n\;a) = a\).
Proof sketch: Induction; step uses lastEl_cons and IH.
Theorem (step_replicate_ident). The all-identity population is a fixed point: \[\mathtt{step}(\mathtt{replicate}\;n\;e) = \mathtt{replicate}\;n\;e.\]
Proof sketch: Use lastEl_replicate_self and stepWith_replicate_ident.
Theorem (iterate_step_add). The semigroup law for iterated diffusion: \[\mathtt{step}^{[n+m]}\;\mathit{xs} = \mathtt{step}^{[n]}(\mathtt{step}^{[m]}\;\mathit{xs}).\]
Proof sketch: Function.iterate_add_apply.
Theorem (iterate_step_replicate_ident). \(\mathtt{step}^{[n]}(\mathtt{replicate}\;L\;e) = \mathtt{replicate}\;L\;e\).
Proof sketch: Induction on n; step applies IH then step_replicate_ident.
Theorem (propagated_ident). An all-identity population propagates as identity forever: \[\mathtt{propagated}(e,\, L,\, n) = \mathtt{replicate}\;L\;e.\]
Proof sketch: Unfolds to iterate_step_replicate_ident.

§4 — Aggregate

The aggregate of a population is the \(\circ\)-fold over all agents: \(\mathtt{aggregate}([x_0,\ldots,x_{k-1}]) = x_0 \circ (x_1 \circ \cdots \circ x_{k-1})\). The aggregate captures the population's combined ideational output.

Theorem (aggregate_nil). \(\mathtt{aggregate}([]) = e\).
Proof sketch: by definition (rfl).
Theorem (aggregate_cons). \(\mathtt{aggregate}(x{::}xs) = x \circ \mathtt{aggregate}(xs)\).
Proof sketch: by definition (rfl).
Theorem (aggregate_replicate_ident). \(\mathtt{aggregate}(\mathtt{replicate}\;k\;e) = e\).
Proof sketch: Induction; step uses id_left and IH.
Theorem (aggregate_append_ident_replicate). Appending identity bystanders does not change the aggregate: \[\mathtt{aggregate}(\mathit{xs} \mathbin{++} \mathtt{replicate}\;k\;e) = \mathtt{aggregate}(\mathit{xs}).\]
Proof sketch: List.foldr_append, then aggregate_replicate_ident.
Theorem (aggregate_replicate_ident_append). \(\mathtt{aggregate}(\mathtt{replicate}\;k\;e \mathbin{++} \mathit{xs}) = \mathtt{aggregate}(\mathit{xs})\).
Proof sketch: Induction on k; step uses id_left and IH.

§5 — Strong-Tie Commutativity

Among agents whose ideas commute — "strong ties" in Granovetter's vocabulary — the order of transmission is aggregate-neutral. The following theorem formalises this.

Theorem (aggregate_swap_adjacent_commute). If \(a \circ b = b \circ a\), then \[\mathtt{aggregate}(\mathit{xs} \mathbin{++} [a, b] \mathbin{++} \mathit{ys}) = \mathtt{aggregate}(\mathit{xs} \mathbin{++} [b, a] \mathbin{++} \mathit{ys}).\] Commuting adjacent ideas does not change the aggregate.
Proof sketch: Induction on xs; base uses associativity and the commutativity hypothesis; step is congruence.

§6 — Idempotent Threshold

An idea \(a\) is idempotent if \(a \circ a = a\). When a population has been fully saturated with an idempotent idea, it is a fixed point: the idea is "tipped."

Theorem (stepWith_replicate_idempotent). If \(a \circ a = a\), then \(\mathtt{stepWith}\;a\;(\mathtt{replicate}\;n\;a) = \mathtt{replicate}\;n\;a\).
Proof sketch: Induction; step uses ha and IH.
Theorem (step_replicate_idempotent). If \(a \circ a = a\), then a fully-saturated idempotent population is a fixed point: \[\mathtt{step}(\mathtt{replicate}(n{+}1)\;a) = \mathtt{replicate}(n{+}1)\;a.\]
Proof sketch: Use lastEl_replicate_self and stepWith_replicate_idempotent.

§7 — Rogers-Style Monotone Adoption

A rule \(f : \mathtt{List}\;\alpha \to \mathtt{List}\;\alpha\) is non-erasing if it never decreases the adoption count: \(\mathtt{adopters}(\mathit{xs}) \le \mathtt{adopters}(f(\mathit{xs}))\) for all \(\mathit{xs}\). The Rogers theorem extends this one-step property to all iterates.

Theorem (rogers_monotone). If \(f\) is non-erasing, then for all populations \(\mathit{xs}\) and \(n \in \mathbb{N}\), \[\mathtt{adopters}(\mathit{xs}) \le \mathtt{adopters}(f^{[n]}(\mathit{xs})).\] Iterating a non-erasing rule yields monotonically non-decreasing adoption.
Proof sketch: Induction on n; step chains IH with the non-erasing property.
Theorem (nonErasing_id). The identity rule is non-erasing.
Proof sketch: le_refl.

§8 — Granovetter Weak-Tie Effect

The Granovetter section proves the micro-mechanism of the "strength of weak ties": a single non-identity agent inserted into an otherwise all-identity population strictly increases adoption after one cyclic transmission step. The proof proceeds by computing step explicitly on this population, then counting adopters on both sides of the inequality.

Theorem (lastEl_replicate_succ). \(\mathtt{lastEl}\;x\;(\mathtt{replicate}(n{+}1)\;a) = a\).
Proof sketch: Induction on n; step uses lastEl_cons and IH applied to a.
Theorem (step_single_a_ident). A single non-identity insertion into an all-identity population duplicates rightward in one step: \[\mathtt{step}(a :: \mathtt{replicate}(j{+}1)\;e) = a :: a :: \mathtt{replicate}\;j\;e.\]
Proof sketch: Unfold step; use lastEl_replicate_succ, id_left, id_right, stepWith_replicate_ident.
Theorem (granovetter_weak_tie). If \(a \ne e\), then \[\mathtt{adopters}(a :: \mathtt{replicate}(j{+}1)\;e) < \mathtt{adopters}(\mathtt{step}(a :: \mathtt{replicate}(j{+}1)\;e)).\] A single non-identity insertion strictly increases adoption after one step.
Proof sketch: Apply step_single_a_ident; compute countP on both sides using hcount; simp with ha.

§9 — Further Structural Lemmas

The final section gathers miscellaneous structural results used elsewhere or stated for completeness.

Theorem (iterate_step_nil). \(\mathtt{step}^{[n]}\;[] = []\).
Proof sketch: Induction; step applies IH then step_nil.
Theorem (step_singleton). A length-1 population maps to its self-composition: \[\mathtt{step}([a]) = [a \circ a].\]
Proof sketch: Unfold step and stepWith; rfl.
Theorem (step_singleton_idempotent). If \(a \circ a = a\), then \(\mathtt{step}([a]) = [a]\).
Proof sketch: Apply step_singleton, then ha.
Theorem (aggregate_replicate_ident_eq). \(\mathtt{aggregate}(\mathtt{replicate}\;k\;e) = e\).
Proof sketch: Delegates to aggregate_replicate_ident.
Theorem (aggregate_append). \(\mathtt{aggregate}(\mathit{xs} \mathbin{++} \mathit{ys}) = \mathtt{List.foldr}\;\mathrm{op}\;(\mathtt{aggregate}\;\mathit{ys})\;\mathit{xs}\).
Proof sketch: Unfold aggregate; apply List.foldr_append.
Theorem (aggregate_replicate_idempotent). If \(a \circ a = a\), then a constant idempotent population aggregates to that element: \[\mathtt{aggregate}(\mathtt{replicate}(n{+}1)\;a) = a.\]
Proof sketch: Induction; base is id_right; step uses IH and ha.

Theoretical Implications

The fixed-point theorems — step_replicate_ident (T12), iterate_step_replicate_ident (T14), and propagated_ident (T15) — prove that every population of ideas has at least one stable attractor: the all-identity configuration. This is not a sociological claim about equilibrium culture; it is a mathematical theorem about the dynamics of the idea monoid. The identity element is the "ground state" of ideational diffusion — the state to which a fully non-adopting population returns. This has a direct implication for theories of cultural change: change requires the presence of at least one non-identity idea. The algebraic structure guarantees that you cannot bootstrap novelty from a homogeneous null population.

The length-conservation theorem (step_length, T8) establishes a fundamental invariant that is surprising when first encountered: diffusion conserves population size exactly. No agents are created or destroyed, no mergers occur, no agents drop out. This corresponds to the closed-world assumption in many diffusion models and makes precise the boundary of the model's applicability. Real-world diffusion involves births and deaths, immigration and emigration. The Idea Theory model is a pure transmission model: it studies what happens to the ideational content of a fixed population under iterated composition. Violations of this invariant — when they occur empirically — are thus diagnostic of forces outside the model.

The Granovetter theorem (granovetter_weak_tie, T28) proves the mechanism behind the "strength of weak ties" from first principles. In a population of \(n+1\) agents where \(n\) hold the identity and one holds \(a \ne e\), one step of cyclic transmission strictly increases adoption. The strictness is crucial: it distinguishes the active role of the lone adopter from the passive stability of the all-identity state. The proof uses only id_left, id_right, and List.countP — no continuous mathematics, no network topology, no probability. The structural result holds for any non-identity idea in any idea-theory structure. This universality is what machine verification guarantees: there are no hidden special-case assumptions.

The aggregate lemmas reveal a deep connection between the local transmission rule and the global structure of the population's combined idea. The aggregate_swap_adjacent_commute theorem (T21) shows that commutativity between adjacent ideas — "strong ties" in Granovetter's vocabulary — is aggregate-neutral. This means that among strongly connected agents (those whose ideas commute), the order of transmission is irrelevant to the population's total output. But weak ties — non-commuting adjacent ideas — do matter: their order affects the aggregate. The algebraic condition for Granovetter's distinction between strong and weak ties is thus commutativity of the idea monoid: strong-tie networks correspond to commutative sub-monoids; weak-tie bridges correspond to non-commuting pairs.

Zoo Connections

Rogers — The S-Curve

Everett Rogers's diffusion of innovations theory (1962, 2003) describes how new ideas spread through populations in an S-shaped curve: slow initial adoption, rapid acceleration once a critical threshold is reached, then deceleration as saturation approaches. The rogers_monotone theorem (T24) formalises the monotonicity half of this picture: any non-erasing transmission rule yields a weakly increasing adoption count over time. Once an agent adopts an idea, they never lose it. The S-curve shape emerges from the interplay between this monotonicity and the population's finite size: adoption must eventually saturate at the population size, which is conserved by step_length (T8). Rogers's key insight — that adoption is a one-way door — is thus a theorem in Idea Theory, not merely an empirical regularity. The formalization makes precise what "non-erasing" means: \(\mathtt{adopters}(\mathit{xs}) \le \mathtt{adopters}(f(\mathit{xs}))\) for every population \(\mathit{xs}\).

Granovetter — The Strength of Weak Ties

Mark Granovetter's seminal paper (1973) argues that "weak ties" — acquaintances rather than close friends — are more important for information diffusion than strong ties, because weak ties bridge otherwise disconnected clusters. The granovetter_weak_tie theorem (T28) formalises the micro-level mechanism: a single non-identity agent inserted into an all-identity population strictly increases adoption after one bilateral transmission step. The lone non-identity agent effectively duplicates: the agent to its left (via cyclic boundary) also acquires the idea. This is the "strength of weak ties" in its sharpest form: one bridging connection is sufficient to strictly increase adoption. The theorem's strict inequality (\(<\)) distinguishes it from the mere monotonicity of Rogers. The aggregate_swap_adjacent_commute lemma (T21) captures the complementary insight that strong ties — commuting adjacent ideas — are aggregate-neutral.

Gladwell — The Tipping Point

Malcolm Gladwell's tipping point theory (2000) argues that social epidemics exhibit threshold behavior: below a critical adoption level, diffusion stalls; above it, it explodes. The step_replicate_idempotent theorem (T23) formalises the maximum-threshold case: when an idempotent idea has reached 100% adoption, the population is a fixed point — the idea is "tipped." The rogers_monotone theorem (T24) shows that adoption is non-decreasing, providing the mechanism for tipping: once past a threshold, diffusion cannot reverse. Gladwell's theory has been criticized for lacking mathematical precision. The Idea Theory formalization provides that precision: tipping points correspond to populations where enough agents hold an idempotent idea that the fixed-point condition of step_replicate_idempotent is met. The granovetter_weak_tie theorem shows that even below the tip, a single adopter in an otherwise non-adopting population initiates strict growth.

Sperber — Epidemiology of Representations

Dan Sperber's epidemiology of representations (Explaining Culture, 1996) models cultural transmission as an "epidemiological" process in which mental representations are selectively transmitted, with cognitive attractors pulling variants toward stable forms. The propagated_ident theorem (T15) captures attractor dynamics in their simplest form: the all-identity population is a global attractor — it cannot be perturbed, because it has no non-identity ideas to diffuse. The step_replicate_idempotent theorem (T23) shows that any idempotent idea forms a local attractor: once fully adopted, it persists. The step_singleton theorem (T30) reveals the micro-mechanism: in a length-1 population, the idea self-composes — it is an attractor precisely when it is idempotent. Sperber's cognitive attractors correspond, algebraically, to idempotent elements of the idea monoid.

Henrich — The Secret of Our Success

Joseph Henrich's cumulative cultural evolution theory (The Secret of Our Success, 2015) emphasises that human culture accumulates improvements through faithful transmission and recombination across generations, constituting a "cultural ratchet." The aggregate theorems formalise the ratchet's neutral element: aggregate_replicate_ident (T18) shows that a population of non-adopters contributes nothing to the cultural aggregate; aggregate_append_ident_replicate (T19) shows that adding silent bystanders changes nothing. The aggregate_replicate_idempotent theorem (T34) shows that a population fully committed to an idempotent idea aggregates to that idea — the ratchet has "clicked." Henrich's key claim — that cultural accumulation is non-erasing — corresponds to rogers_monotone: each ratchet click is irreversible. The formalization makes explicit what the metaphor leaves implicit: "non-erasing" is a property of the transmission rule, not of the ideas themselves.

Bass — Diffusion Model

Frank Bass's parametric diffusion model (1969) describes adoption through two channels: "innovators" who adopt independently (rate \(p\)) and "imitators" who adopt by contact with existing adopters (rate \(q\)). The step_single_a_ident lemma (T27) formalises the imitator mechanism in its purest form: a single adopter in a population of non-adopters generates one new adopter per step, corresponding to \(q=1\) in Bass's model. The length-preservation theorem (step_length, T8) shows that the model conserves population size — the total market is fixed, as Bass assumes. The iterate_step_length theorem (T9) extends this to all iterates: at every time step, the population count is conserved. Bass's model predicts eventual saturation; the Idea Theory formalization shows that the algebraic mechanism — idempotent fixed points — guarantees this for any idempotent idea under the step_replicate_idempotent theorem.

Proof Architecture

The 34 theorems are not independent: they form a dependency lattice in which the definitional lemmas of §1 underpin every later result. The diagram below traces the main proof lines.

Dependency chains

  • Length conservation chain: stepWith_cons → stepWith_length → step_length → iterate_step_length. Each result lifts the previous one one level of abstraction: from a single cons-cell, to a single step, to all iterates.
  • Fixed-point chain: lastEl_replicate_self + stepWith_replicate_ident → step_replicate_ident → iterate_step_replicate_ident → propagated_ident. The chain proceeds by first establishing the fixed-point at the level of stepWith, then lifting to step, then to all iterates, then to the named propagated combinator.
  • Granovetter chain: lastEl_replicate_succ → step_single_a_ident → granovetter_weak_tie. The first lemma computes the cyclic boundary element; the second computes the full output of step on the target population; the third counts adopters on both sides and closes with the strict inequality.
  • Idempotent chain: stepWith_replicate_idempotent + lastEl_replicate_self → step_replicate_idempotent → step_singleton_idempotent → aggregate_replicate_idempotent. Idempotence propagates from the local transmission lemma through to the global aggregate.

Proof tactics

The proofs make heavy use of Lean 4's simp and induction tactics, together with the monoid axioms (id_left, id_right, mul_assoc) and list lemmas (List.foldr_append, List.length_cons). No proof requires more than 10 tactic lines. The most involved result is granovetter_weak_tie, which needs List.countP_cons and a case split on a = ident discharged by ha.

Decidability hypotheses

The Rogers and Granovetter theorems require [DecidableEq α] because adopters is defined via List.countP (· ≠ ident), which needs decidable equality to evaluate. All structural lemmas (length, fixed points, aggregate) work in fully general monoids with no decidability assumption.

The idea monoid \((\mathcal{I}, \circ, e)\) is assumed throughout to satisfy the standard monoid axioms: associativity \((a \circ b) \circ c = a \circ (b \circ c)\), left identity \(e \circ a = a\), and right identity \(a \circ e = a\). No commutativity is assumed globally; commutativity hypotheses are stated explicitly where needed (as in aggregate_swap_adjacent_commute).
