MeaningGames

Lean 4 Extension — Wittgenstein meets Nash: use-theory of meaning as coordination equilibrium

Module: IdeaTheory.Extensions.MeaningGames  ·  Theorems/Lemmas: 25  ·  Zoo entries: 7  ·  Source: GitHub

Overview

The MeaningGames extension formalizes the intersection of two landmark 20th-century accounts of linguistic meaning. A meaning game is a coordination game played by \(n\) speakers over an idea monoid \((\alpha, \circ, e)\): each speaker holds a use \(\sigma(i) \in \alpha\), and their joint practice produces the effective meaning \[ \mathrm{effectiveMeaning}(\sigma) \;=\; \sigma(0) \circ \sigma(1) \circ \cdots \circ \sigma(n{-}1), \] computed as a left-fold over the monoid operation starting from the identity \(e\).

A profile \(\sigma\) is a convention (\(\mathrm{IsConvention}(G,\sigma)\)) when no speaker benefits from unilateral deviation — the Lewis-Nash equilibrium condition. The coordination payoff \(\mathrm{coordPayoff}(n,M)(i,\sigma)\) equals \(1\) when \(\mathrm{effectiveMeaning}(\sigma) \in M\), else \(0\).

The module’s capstone result, wittgenstein_nash_existence, proves that for any non-empty finite meaning set \(M\) and any \(n \ge 1\), an effective convention \(\sigma\) exists. This is the formal bridge between Wittgenstein’s use-theory of meaning (PI §43) and Lewis’s analysis of conventions (1969) as Nash equilibria of coordination games.

The module covers 11 thematic sections: effective meaning lemmas (L1–L6), coordination payoff properties (L7–L9), coordination game conventions (T1–T4), refinement and closure (T5–T8), private language (T9), Lewis salience (T10), family resemblance (T11–T12), strict dominance (T13), the Kripkenstein paradox (T14), iterated meaning games (T15), and aggregation (T16).

Core Definitions

MeaningGame structure and effectiveMeaning

The central types are a MeaningGame record packaging speaker count, meaning predicate, and payoff function; effectiveMeaning collapsing a speaker profile via a monoid left-fold; and IsConvention encoding the Nash no-deviation condition. 

-- MeaningGame and core definitions
structure MeaningGame (α : Type u) [IdeaTheoryStructure α] where
  n : ℕ
  meaningSet : α → Prop
  payoff : Fin n → (Fin n → α) → ℝ

def effectiveMeaning {n : ℕ} (σ : Fin n → α) : α :=
  (List.ofFn σ).foldl op ident

def IsConvention (G : MeaningGame α) (σ : Fin G.n → α) : Prop :=
  ∀ (i : Fin G.n) (b : α), G.payoff i (update σ i b) ≤ G.payoff i σ

The effectiveMeaning function converts the profile \(\sigma : \mathrm{Fin}\,n \to \alpha\) into a list via List.ofFn and left-folds with the monoid operation op, starting from the identity ident. IsConvention is the Nash condition: for every speaker \(i\) and every alternative use \(b\), deviating from \(\sigma\) does not improve \(i\)’s payoff.

Coordination payoff and game

-- Coordination payoff and coordination game
noncomputable def coordPayoff (n : ℕ) (M : α → Prop) :
    Fin n → (Fin n → α) → ℝ :=
  fun _ σ => by classical exact if M (effectiveMeaning σ) then 1 else 0

noncomputable def coordGame (n : ℕ) (M : α → Prop) : MeaningGame α :=
  { n := n, meaningSet := M, payoff := coordPayoff n M }

The coordination game rewards all speakers equally with payoff \(1\) when the joint meaning lands in \(M\), and \(0\) otherwise. The noncomputable annotation arises from the classical if-then-else on a Prop; the classical tactic recovers decidability.

Private language and family resemblance

-- Private language and family resemblance
def FamilyResemblance (F : α → Prop) : Prop :=
  ∀ a, F a → ∀ b, F b → ∃ c : α, op c c = c ∧ op c a = a ∧ op c b = b

theorem private_no_unique_convention (M : α → Prop) (σ : Fin 1 → α) :
    IsConvention ({ n := 1, meaningSet := M,
                   payoff := trivialPayoff 1 } : MeaningGame α) σ

Kripkenstein finite underdetermination

-- Kripkenstein finite underdetermination
theorem kripkenstein_underdetermination {a b : α} (hab : a ≠ b)
    (h : List α) :
    ∃ r₁ r₂ : List α → α, r₁ h = r₂ h ∧ r₁ ≠ r₂

Wittgenstein-Nash bridge (capstone)

-- The Wittgenstein-Nash bridge (capstone theorem)
theorem wittgenstein_nash_existence {n : ℕ} (hn : 1 ≤ n) {M : α → Prop}
    {a : α} (ha : M a) :
    ∃ σ : Fin n → α, IsConvention (coordGame n M) σ

Theorem Catalogue

§1 — Effective Meaning Lemmas (L1–L6)

Theorem (effectiveMeaning_zero). \(\mathrm{effectiveMeaning}(\sigma : \mathrm{Fin}\,0 \to \alpha) = e.\) With no speakers, joint practice yields the identity — there is no language without language users. The zero-player game is the formal image of absolute linguistic silence.
Proof sketch: simp with effectiveMeaning unfolded to List.ofFn 0 = [], so the fold over the empty list returns the identity \(e\).
Theorem (effectiveMeaning_succ). \(\mathrm{effectiveMeaning}(\sigma : \mathrm{Fin}(n{+}1) \to \alpha) = \bigl(\mathrm{List.ofFn}(\lambda i,\, \sigma(i.\mathrm{succ}))\bigr).\mathrm{foldl}\,(\circ)\,(\sigma\,0).\) The effective meaning unfolds recursively: the first speaker’s use becomes the initial accumulator, and the remaining speakers compose into it left-to-right.
Proof sketch: Unfold effectiveMeaning; apply List.ofFn_succ to split the list head from the tail; use id_left to absorb the initial identity fold step.
Theorem (effectiveMeaning_one). \(\mathrm{effectiveMeaning}(\sigma : \mathrm{Fin}\,1 \to \alpha) = \sigma\,0.\) In a monologue the effective meaning is the sole speaker’s use. This lemma is the formal basis of the private-language argument: a single speaker’s “effective meaning” is just their own use, with no intersubjective grounding.
Proof sketch: Apply effectiveMeaning_succ; simp — the tail list List.ofFn (Fin 0 → α) is empty, so the fold returns the accumulator \(\sigma\,0\) unchanged.
Theorem (effectiveMeaning_const_ident). \(\mathrm{effectiveMeaning}(\lambda\,\_:\mathrm{Fin}\,n,\; e) = e.\) If every speaker plays the identity, joint practice produces the identity. The constant-identity profile is the algebraic image of a community with nothing to say: their aggregate use is indistinguishable from no use at all.
Proof sketch: Unfold effectiveMeaning; apply foldl_op_ident_list — every element of the generated list is \(e\), and each fold step is \(e \circ e = e\).
Theorem (effectiveMeaning_const_idemp). If \(c \circ c = c\) and \(n \ge 1\), then \(\mathrm{effectiveMeaning}(\lambda\_,\, c) = c.\) A constant idempotent profile yields that idempotent as effective meaning. Algebraically, an idempotent \(c\) absorbs all repetition of itself: \(c^n = c\) for all \(n \ge 1\). This is the formal content of Schelling focal-point stability.
Proof sketch: Apply effectiveMeaning_succ; then foldl_op_const_of_idemp using hypothesis ha : c ∘ c = c on each tail element, driving the fold to \(c\).
Theorem (effectiveMeaning_loudProfile). If \(n \ge 1\), then \(\mathrm{effectiveMeaning}(\mathrm{loudProfile}\,n\,a) = a.\) In the loud-speaker profile (speaker \(0\) says \(a\), everyone else says \(e\)), the effective meaning is \(a\). This is the key witness for the Wittgenstein-Nash bridge: one sufficiently “loud” speaker can fix a convention even in a large community.
Proof sketch: Apply effectiveMeaning_succ. The tail consists entirely of \(e\), so foldl_op_ident_list returns the initial accumulator \(a\) unchanged.

§2 — Coordination Payoff (L7–L9)

Theorem (coordPayoff_eq_one_iff). \(\mathrm{coordPayoff}(n,M)(i,\sigma) = 1 \;\iff\; M(\mathrm{effectiveMeaning}(\sigma)).\) The coordination payoff equals \(1\) exactly when the joint meaning lies in the meaning set. This is the formal criterion for successful linguistic coordination: the community’s aggregate use achieves the target.
Proof sketch: Classical split; split_ifs on \(M(\mathrm{effectiveMeaning}(\sigma))\); both directions follow immediately from the if-then-else definition of coordPayoff.
Theorem (coordPayoff_le_one). \(\mathrm{coordPayoff}(n,M)(i,\sigma) \le 1.\) Coordination can never yield more than perfect alignment — the Nash-style upper bound on payoffs. This ensures that conventions are characterized by achieving the maximum, not exceeding it.
Proof sketch: split_ifs; in the true branch le_refl 1; in the false branch norm_num gives \(0 \le 1\).
Theorem (coordPayoff_nonneg). \(0 \le \mathrm{coordPayoff}(n,M)(i,\sigma).\) Coordination never hurts — the Lewis no-punishment assumption. Every speaker receives a non-negative payoff regardless of the profile, ensuring that participation in the meaning game is individually rational.
Proof sketch: split_ifs; norm_num closes both cases since the payoff is either \(0\) or \(1\).

§3 — Conventions in the Coordination Game (T1–T4)

Theorem (coord_convention_of_meaning). If \(M(\mathrm{effectiveMeaning}(\sigma))\), then \(\sigma\) is a convention of \(\mathrm{coordGame}(n,M)\). Any profile whose joint meaning already lies in \(M\) is a convention — successful use is the convention. This is the Wittgenstein use-theory principle in its sharpest form: there is no gap between correct use and conventional use.
Proof sketch: Show coordPayoff = 1 for \(\sigma\) via coordPayoff_eq_one_iff; then coordPayoff_le_one bounds every deviation from above by \(1\).
Theorem (coord_convention_zero). The empty profile is vacuously a convention of \(\mathrm{coordGame}(0,M)\). A language community with zero speakers trivially satisfies the convention condition. This degenerate case underscores that conventions are properties of communities, not individuals: with no community there is no non-trivial convention.
Proof sketch: intro i _; exact i.elim0 — there are no speakers, so the universal quantifier is vacuously satisfied.
Theorem (wittgenstein_nash_existence). If \(n \ge 1\) and \(M(a)\), then \(\exists\,\sigma : \mathrm{Fin}\,n \to \alpha,\; \mathrm{IsConvention}(\mathrm{coordGame}(n,M),\sigma).\) THE CAPSTONE THEOREM. For any non-empty meaning set and any \(n \ge 1\), a Nash equilibrium convention exists. This is the Wittgenstein-Nash bridge: use-theory and convention theory are two descriptions of the same algebraic reality.
Proof sketch: Witness \(\mathrm{loudProfile}\,n\,a\); apply coord_convention_of_meaning; rewrite with effectiveMeaning_loudProfile to get \(M(a)\), which is exactly ha.
Theorem (exists_convention_one_speaker). If \(M(a)\), then \(\exists\,\sigma : \mathrm{Fin}\,1 \to \alpha,\; \mathrm{IsConvention}(\mathrm{coordGame}(1,M),\sigma).\) A one-speaker meaning game with non-empty meaning set has a convention. Even a monologue can instantiate a Lewis convention, though as private_no_unique_convention shows, the convention is not unique when payoffs are trivial.
Proof sketch: Witness \(\lambda\,\_\Rightarrow a\); apply coord_convention_of_meaning; use effectiveMeaning_one to reduce effective meaning to \(a\), then close with ha.

§4 — Refinement and Closure (T5–T8)

Theorem (coord_convention_mono). If \(M \subseteq N\) and \(\sigma\) is an \(M\)-convention, then \(\sigma\) is an \(N\)-convention. Enlarging the meaning set preserves conventions — meaning compatibility is monotone. Any convention for a more restrictive meaning set automatically satisfies the requirements of a more permissive one.
Proof sketch: Apply coord_convention_of_meaning; use hMN to transport \(M(\mathrm{effectiveMeaning}(\sigma))\) to \(N(\mathrm{effectiveMeaning}(\sigma))\).
Theorem (coord_convention_union). If \(M(\mathrm{effectiveMeaning}(\sigma)) \lor N(\mathrm{effectiveMeaning}(\sigma))\), then \(\sigma\) is a \((M \lor N)\)-convention. The union of two meaning sets inherits conventions coordinated under either. Wittgenstein’s family resemblance: overlapping sets share conventional profiles.
Proof sketch: Apply coord_convention_of_meaning with the disjunction directly; the union predicate \((M \lor N)(x)\) is defined as \(M(x) \lor N(x)\).
Theorem (effectiveMeaning_in_meaningSet). If \(e \in M\) and \(M\) is closed under \(\circ\), then the effective meaning of any all-meaningful profile lies in \(M\). Inferential closure of conceptual content: if all uses are meaningful, the joint practice is meaningful. This is the algebraic form of Davidson’s holism — globally coherent meaning attributions form a sub-monoid.
Proof sketch: foldl_op_in_M with hOp : ∀ x y, M x → M y → M (x∘y) and hId : M e, applied inductively along the list of speaker uses.
Theorem (convention_of_constant_payoff). If payoffs are profile-independent, then every profile is a convention. The game-theoretic analogue of radical charity: when payoffs don’t depend on play, all profiles are Nash equilibria. This is the trivial game in which coordination imposes no constraint.
Proof sketch: intro i b; le_of_eq using h i σ b — profile-independence means any deviation has the same payoff as the original.

§5 — Private Language (T9)

Theorem (private_no_unique_convention). In a one-speaker trivial-payoff game, every profile is a convention. The private-language argument: with no coordination signal, any use is vacuously conventional, so meaning cannot be fixed privately. Wittgenstein’s point is not that private language is impossible but that it fails to privilege any particular use as the correct one — there is no fact of the matter.
Proof sketch: intro i b; unfold trivialPayoff; le_refl — all payoffs are equal (e.g., \(0\)), so no deviation can strictly improve.

§6 — Lewis Salience and Focal Points (T10)

Theorem (salient_convention_in_meaning). If \(a \circ a = a\) and \(M(a)\) and \(n \ge 1\), then \((\lambda\_,\, a)\) is a convention of \(\mathrm{coordGame}(n,M)\). Salient (idempotent) uses that lie in the meaning set form self-enforcing conventions — Schelling focal points. The algebraic condition \(a \circ a = a\) captures the intuition that a focal point is one that “reproduces itself” under aggregation: unanimous use of \(a\) yields \(a\).
Proof sketch: Apply coord_convention_of_meaning; rewrite with effectiveMeaning_const_idemp using salience hypothesis hs : a∘a=a and hn : 1 ≤ n; close with hM.

§7 — Family Resemblance (T11–T12)

Theorem (family_resemblance_idempotent). If \(F\) has family resemblance, then \(\forall a \in F,\; \exists c,\; c \circ c = c \;\wedge\; c \circ a = a.\) Every member of a family-resemblance class has an idempotent “common feature” that absorbs it. The idempotent \(c\) is the shared structural element that explains why \(a\) belongs to the family — not a necessary-and-sufficient condition but a common structural anchor.
Proof sketch: Apply hF a ha a ha (setting \(b := a\)); extract \(c\) from the existential; return (c, hc.1, hc.2.1).
Theorem (family_resemblance_convention). If \(F\) has family resemblance and \(F(a)\) and \(n \ge 1\), then \(\exists\,\sigma,\; \mathrm{IsConvention}(\mathrm{coordGame}(n,F),\sigma).\) Family resemblance is a sufficient condition for linguistic coordination. Wittgenstein’s anti-essentialism is compatible with Lewis’s convention theory: concepts without necessary-and-sufficient conditions can still ground stable conventions.
Proof sketch: Delegates directly to wittgenstein_nash_existence hn ha — no additional structure beyond the non-emptiness of \(F\) at \(a\) is needed for the existence proof.

§8 — Strict Dominance (T13)

Theorem (dominance_unique). If \(u\) strictly dominates every alternative for every speaker, then every convention has every speaker playing \(u\). Strict dominance implies convention uniqueness — there is exactly one “correct” use. When one use strictly dominates all others, Lewis’s alternative conventions collapse to a single point: the dominant use.
Proof sketch: by_contra h — assume some speaker \(i\) plays \(b \ne u\); apply hd i b to get a strict payoff inequality for deviating to \(u\); rw update_eq_self if \(b = u\); combine with the convention inequality via linarith.

§9 — Kripkenstein (T14)

Theorem (kripkenstein_underdetermination). For any finite history \(h\) and any \(a \ne b\), there exist two distinct rules \(r_1 \ne r_2\) that both map \(h\) to the same value: \[ \exists\, r_1\, r_2 : \mathrm{List}\,\alpha \to \alpha,\quad r_1\,h = r_2\,h \;\wedge\; r_1 \ne r_2. \] Finite observation can never single out a rule — the rule-following paradox in its sharpest finite form. This is Kripke’s sceptical argument as a theorem: not merely a philosophical possibility but a provable mathematical fact.
Proof sketch: Witness \(r_1 = \mathrm{const}\,a\) and \(r_2 = (\lambda l,\; \mathrm{if}\; l = h\; \mathrm{then}\; a\; \mathrm{else}\; b)\). Both agree on \(h\) (output \(a\)); they differ on \(h \mathbin{++} [e]\) by a list-length argument together with hab : a ≠ b.

§10 — Iterated Meaning Games (T15)

Theorem (iterMeaning_monotone). \(\mathrm{iterMeaning}(M,k,a) \implies \mathrm{iterMeaning}(M,k{+}1,a).\) Cultural drift is monotone — every meaning available at step \(k\) remains available at step \(k{+}1\). This is the formal content of the “language-as-tradition” view: a linguistic community cannot simply erase meanings that have been established. Meanings accumulate; they do not evaporate.
Proof sketch: Or.inl carries the inductive step: if \(a\) is reachable at step \(k\) via the inductive hypothesis, it is also reachable at step \(k+1\) by left-inclusion in the disjunction.

§11 — Aggregation (T16)

Theorem (aggregation_constant_convention). If \(n \ge 1\), \(a \circ a = a\), and \(M(a)\), then \(M(\mathrm{effectiveMeaning}(\lambda\_,\, a)).\) A unanimous salient convention realizes perfect coordination — the agreed reading of a unanimous convention lies in the meaning set. This closes the loop between individual salience, collective coordination, and successful meaning.
Proof sketch: Rewrite with effectiveMeaning_const_idemp ha hn to reduce the effective meaning to \(a\); then exact hM.

Philosophical Zoo

Seven traditions in the philosophy of language, formalized through the lens of MeaningGames.

Wittgenstein I — Tractarian Meaning

The early Wittgenstein (Tractatus Logico-Philosophicus, 1921) held that meaning is pictorial: propositions are logical pictures of facts, and meaning is determined by the correspondence between propositional structure and worldly structure. On this view, meaning is fully determined by the internal structure of an idea — independent of use. The effectiveMeaning_const_ident theorem (L4) implicitly refutes this: a profile of nothing but identities — ideas with no pictorial content — yields the identity, i.e., no meaning. Meaning requires non-identity content. But the early Wittgenstein would insist that the content of that identity is still determined by its logical form, not its use. The contrast with Wittgenstein II is stark: the Tractatus view would predict that effectiveMeaning is determined by the logical form of each \(\sigma(i)\), independently. The Idea Theory model makes no such assumption — effective meaning is a joint product of all speakers’ uses, not a function of individual logical form alone.

Wittgenstein II — Meaning as Use

The later Wittgenstein (Philosophical Investigations, 1953) abandoned the picture theory in favor of the use-theory: “the meaning of a word is its use in the language” (PI §43). A word has meaning only within a language game — a form of life. The effectiveMeaning function formalizes this exactly: meaning is the \(\circ\)-aggregate of uses, not any intrinsic property of the idea. The effectiveMeaning_one theorem (L3) confirms the private-language argument’s formal structure: a lone speaker’s “effective meaning” is just their own use — there is no intersubjective grounding. But wittgenstein_nash_existence (T3) extends this: when multiple speakers coordinate, a genuine convention emerges. Wittgenstein’s language games are formalized as coordination games, and his criterion for meaning — successful coordination — is exactly the Lewis condition for convention.

Lewis — Convention

David Lewis (Convention: A Philosophical Study, 1969) defined a convention as a Nash equilibrium of a coordination game with the properties that (a) it is common knowledge, (b) any deviation would be self-defeating, and (c) an alternative convention could have served equally well. The IsConvention predicate formalizes Lewis’s Nash equilibrium condition exactly: no unilateral deviation improves payoff. The wittgenstein_nash_existence theorem proves Lewis’s existence claim: for any non-empty meaning set, a convention exists. The coord_convention_mono theorem (T5) formalizes Lewis’s counterfactual condition: if a convention works for meaning set \(M\), it works for any superset \(N\). dominance_unique (T13) formalizes the case where Lewis’s alternative conventions collapse to one — when a single use strictly dominates all others. Lewis’s framework is thus a special case of the more general Idea Theory coordination game structure.

Kripkenstein — Rule-Following Scepticism

Saul Kripke’s interpretation of Wittgenstein (Wittgenstein on Rules and Private Language, 1982) presents the “sceptical paradox”: no fact about a speaker’s past behavior or mental states can determine which rule they are following, since any finite history is compatible with infinitely many rules. The kripkenstein_underdetermination theorem (T14) gives the precise mathematical content of this claim: for any finite observation history \(h\), there exist two distinct rules that agree on \(h\) but disagree on extensions. The proof constructs the two rules explicitly: the constant rule and the “deviant” rule that agrees on \(h\) but produces \(b \ne a\) elsewhere. Kripkenstein’s paradox is thus a theorem, not merely a philosophical argument. But wittgenstein_nash_existence shows that conventions still exist despite underdetermination: we cannot determine which rule a speaker follows, but we can establish that a convention exists.

Schelling — Focal Points

Thomas Schelling (The Strategy of Conflict, 1960) argued that among multiple Nash equilibria, agents converge on “salient” or “focal” ones — points that stand out by virtue of some contextual prominence rather than payoff structure. The salient_convention_in_meaning theorem (T10) formalizes Lewis’s interpretation of Schelling: an idempotent idea \(a\) that lies in the meaning set is a self-enforcing focal point — playing it unanimously is a convention. The salience condition — Salient a means \(a \circ a = a\) — is the algebraic criterion for a “stable” use: an idea that is self-reinforcing under composition. Schelling’s focal points are thus the idempotent elements of the idea monoid that lie in the meaning set. The family_resemblance_idempotent theorem (T11) shows that family resemblance classes always contain such focal points — shared idempotent “common features” that serve as natural coordination points.

Davidson — Radical Interpretation

Donald Davidson (Inquiries into Truth and Interpretation, 1984) argued that meaning must be understood through radical interpretation: we attribute meanings to speakers by assuming they are largely rational and mostly true. The convention_of_constant_payoff theorem (T8) captures the trivial case of Davidson’s principle: when payoffs don’t depend on the profile, every use is equally conventional — the linguistic equivalent of Davidson’s “principle of charity” at its most permissive. The effectiveMeaning_in_meaningSet theorem (T7) captures the closure condition that Davidson’s holism requires: if all uses are meaningful (lie in \(M\)) and \(M\) is closed under composition, the joint meaning stays within \(M\). Davidson’s “holism of the mental” — the claim that meaning attributions must be globally coherent — corresponds algebraically to the closure of the meaning set under the composition operation.

Grice — Meaning and Intention

Paul Grice (Meaning, 1957; Logic and Conversation, 1975) distinguished natural meaning (smoke means fire) from non-natural meaning (the speaker means something by an utterance) and argued that non-natural meaning involves complex iterated intentions. The coord_convention_of_meaning theorem (T1) gives the non-intentional analogue: meaning is not defined by what the speaker intends, but by whether their use achieves coordination. A profile is conventional if and only if no speaker benefits by deviating — no reference to intentions. The kripkenstein_underdetermination theorem (T14) shows that intentions — even infinite hierarchies of them — cannot determine meaning, because finitely many observed uses are compatible with infinitely many intention-realizing rules. Grice’s intentional theory of meaning is thus underdetermined in exactly the way Kripkenstein predicts. The Idea Theory convention-based account is immune: conventions are determined by Nash equilibria, not by intentions.

Implications

The Wittgenstein-Nash bridge theorem (wittgenstein_nash_existence) is not merely a technical result in applied game theory. It is the formal proof that the two most influential 20th-century accounts of linguistic meaning — Wittgenstein’s use-theory and Lewis’s convention theory — are theorems about the same mathematical structure. Wittgenstein said meaning is use; Lewis said conventions are Nash equilibria of coordination games. The bridge theorem says: if meaning is use (effectiveMeaning is the \(\circ\)-aggregate of uses), then for any non-empty meaning set, a Lewis convention exists. Use-theory and convention-theory are not competing accounts but complementary aspects of a single algebraic structure. The machine-verified proof leaves no room for doubt about the connection.

The Kripkenstein underdetermination theorem (kripkenstein_underdetermination) proves that the rule-following paradox is a genuine theorem, not a philosophical sleight of hand. For any finite observation history, two distinct rules always agree on that history. This means that no amount of finite behavioral observation can uniquely determine a speaker’s rule. But the Kripkenstein paradox is usually taken to threaten the very existence of meaning. The wittgenstein_nash_existence theorem deflates this threat: even though rules are underdetermined by finite data, conventions still exist. Meaning does not require a uniquely determined rule — it requires only a stable coordination pattern. The Kripkenstein paradox shows that meaning cannot be reduced to rule-following; the Wittgenstein-Nash bridge shows that meaning is convention, and conventions exist regardless of which rule the speakers are “really” following.

The private-language argument (private_no_unique_convention, T9) is formalized as the observation that in a single-speaker game with trivial payoffs, every profile is vacuously a convention. This means that private language is not impossible — it is meaningless in the technical sense: any use is equally conventional when there is no coordination game to be won. Wittgenstein’s argument is not that a private linguist cannot follow rules, but that there is no fact of the matter about which rule they are following, because there is no intersubjective coordination game to fix a convention. The theorem makes this precise: without at least two speakers and a non-trivial payoff structure, “convention” has no content.

The family resemblance theorems (family_resemblance_idempotent, T11; family_resemblance_convention, T12) formalize Wittgenstein’s revolutionary insight about concepts (PI §§65–67): natural language concepts are not defined by necessary and sufficient conditions but by overlapping similarities — “a complicated network of similarities overlapping and criss-crossing.” The algebraic formalization is: a family resemblance class \(F\) has the property that any two members share a common idempotent “feature” \(c\) such that \(c\) absorbs both (\(c \circ a = a\) and \(c \circ b = b\)). The family_resemblance_convention theorem shows that this structural condition is sufficient for a convention to exist — family resemblance is not just a description of concepts but a sufficient condition for coordination. Wittgenstein’s anti-essentialism is thus compatible with Lewis’s coordination-theoretic account of conventions.

The aggregation and monotonicity theorems reveal the temporal dimension of meaning. iterMeaning_monotone (T15) shows that cultural transmission is non-erasing: meanings available at step \(k\) remain available at step \(k{+}1\). This is the formal content of the “language-as-tradition” view: a linguistic community cannot simply erase meanings that have been established. The aggregation_constant_convention theorem (T16) shows that when all speakers agree on a salient use and it lies in the meaning set, perfect coordination is achieved. These two results together describe the lifecycle of a convention: it is established through coordination (Wittgenstein-Nash bridge), propagates non-erasingly through time (iterMeaning_monotone), and sustains perfect coordination as long as the unanimous salient use remains in the meaning set.
